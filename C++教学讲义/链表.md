# 链表
## 引入
链表的特点是：用一组任意的存储单元存储线性表的数据元素（这组存储单元可以是连续的，也可以不连续）。通过如链条一般的指针来连接元素。它的特点是插入与删除数据十分方便，但寻找与读取数据的表现欠佳。常用的链表操作有：初始化、添加、遍历、插入、删除、查找、排序、释放等。
